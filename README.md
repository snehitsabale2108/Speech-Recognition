# Speech-Recognition
A simple Python-based project that converts spoken words into text using speech recognition technology. It captures voice input through a microphone and processes it in real time, making it useful for voice commands, virtual assistants, and hands-free applications.

## Features

- Converts speech into text
- Real-time voice recognition
- Easy and beginner-friendly project
- Supports microphone input
- Fast and accurate speech processing

---

## Technologies Used

- Python
- Python IDLE
- SpeechRecognition Library
- PyAudio

---

## How to Run
### Install Required Libraries

```bash
pip install SpeechRecognition
pip install pyaudio
```

---

### Run the Program

```bash
python speech_recognition.py
```

---

### Speak Through Microphone

The program will listen to your voice and convert it into text.
